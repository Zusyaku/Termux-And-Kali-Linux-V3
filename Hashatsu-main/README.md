# Hashatsu - Hash Killer
Simple hash killer with some options

## How to Use
```sh
python hash.py -m <mode> -s <hash>
```

**Example :**

```sh
python hash.py -m brute -s ae2b1fca515949e5d54fb22b8ed95575
```

**Or :**
```sh
python hash.py -s ae2b1fca515949e5d54fb22b8ed95575
```
